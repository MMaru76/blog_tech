# 【Amazon Linux2】 AWS CLI 2 セットアップ方法

CLI 2を入れる利点としては､出力形式をJSONだけではなく､YAMLも対応されている

# 前提条件

  • unzipなどのコマンドが利用前提

# パッケージ更新

パッケージ更新後に再起動を実施

$ sudo yum -y update
$ sudo reboot
1
2

# AWS CLI 2 インストール

$ curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
$ unzip awscliv2.zip
$ sudo ./aws/install
