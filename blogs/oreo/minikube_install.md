## 実行環境

|項目|値|
|:--:|:--:|
|型式|Mac mini (2018)|
|プロセッサ|3.2 GHz 6コアIntel Core i7|
|メモリ|64 GB 2667 MHz DDR4|
|グラフィックス|Intel UHD Graphics 630 1536 MB|

## 簡単セットアップ

```bash
$ brew update
$ brew install hyperkit
$ brew install minikube
$ minikube version
```
